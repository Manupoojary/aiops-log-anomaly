⏹️ HOW TO STOP THE PROJECT
Step 1️⃣ Stop Python Processes

In Terminal 1 (pipeline):

Ctrl + C


Kafka producer already exits automatically.

Step 2️⃣ Stop Docker Containers

From project root:

docker compose down


✔ Stops Kafka, Elasticsearch, Grafana
✔ Data remains (unless -v is used)

🧹 CLEAN STOP (Optional – Full Reset)

Use this only if you want a fresh start:

docker compose down -v
